# iot_surf_clock
IOT surf clock with weather, surf and water quality forecasts.

![Display preview](server/preview.png)

## FEATURES

- GPIO Access for I/O
    - E-paper 800x480 7.5 inch display
    - Relay circuit for turning light on / off
    - Microphone and speaker for voice commands - may need amplifier

- API Access
    - Build API for home server to access weather and surf data
    - Weather reading
    - Maybe daily news and stocks updates? 
    - Alarm data
- Chron jobs
    - Wakeup routine triggering different functionality
    - bedtime routine?

## Hardware 

- Raspberry Pi Zero 2w
